# Github blog
Trying to make custom static blog engine-not jekyll.

##개발 방향
-> md파일들은 따로 모아둔다.
-> 헤더, 푸터 등 역시 따로 구현한다.
-> 블로그에 표시되는 파일은 전부 같은 파일인데, 오직 가리키는 포스트의 주소만 다르다. 나머지는 전부 javascript로 구현하자.
-> 이렇게 하면 로컬에서 테스트하기가 편리하다. Bracket 하나만 가지고도 테스트가 가능하므로.
